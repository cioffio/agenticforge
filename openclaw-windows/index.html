Come installare OpenClaw su Windows senza impazzire

La documentazione ufficiale raccomanda di usare OpenClaw su Windows tramite WSL2. Questo ti evita molte incompatibilita tipiche del setup nativo e rende il flusso piu vicino a quello usato su Linux.

Strada consigliata: WSL2

Se parti da zero, installa WSL2 con Ubuntu e lavora li dentro. Una volta aperta la shell Linux, usa lo script ufficiale:

curl -fsSL --proto '=https' --tlsv1.2 https://openclaw.ai/install.sh | bash
openclaw onboard --install-daemon
openclaw dashboard

Questo percorso e il piu affidabile per chi vuole usare CLI, gateway e plugin senza attrito inutile.

Alternativa: PowerShell nativo

Se vuoi provare il setup nativo su Windows, il comando ufficiale PowerShell e questo:

iwr -useb https://openclaw.ai/install.ps1 | iex

Resta pero la via meno raccomandata: puo andar bene per test rapidi, ma in generale WSL2 resta la scelta piu stabile.

Subito dopo l'installazione

  • Esegui l'onboarding guidato.
  • Apri la dashboard per testare chat e controllo UI.
  • Se qualcosa non torna, usa un setup piu pulito dentro WSL2 invece di forzare workaround su PowerShell.
