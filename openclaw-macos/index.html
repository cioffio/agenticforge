Come installare OpenClaw su macOS

Su macOS la via piu semplice resta lo script ufficiale. Se vuoi arrivare rapidamente alla prima chat e poi al primo agente utile, il flusso giusto e installazione, onboarding e test finale via dashboard.

Comando ufficiale

curl -fsSL --proto '=https' --tlsv1.2 https://openclaw.ai/install.sh | bash

Lo script gestisce installazione e prerequisiti di base. E la scelta piu diretta se vuoi evitare setup frammentati.

Passi successivi

openclaw onboard --install-daemon
openclaw dashboard
  • L'onboarding configura le impostazioni iniziali in modo guidato.
  • La dashboard ti permette di verificare subito che il sistema risponda.
  • Da li puoi poi aggiungere agenti, plugin e tool utili al tuo workflow.

Quando questa guida basta e quando no

Se vuoi solo installare OpenClaw e arrivare a una prima chat, questa pagina ti basta. Se invece vuoi collegare modelli, permessi, plugin o un agente pronto per task concreti, conviene passare a una configurazione guidata piu completa.
