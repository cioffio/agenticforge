دليل التثبيت احجز الإعداد

الوكلاء

كيفية تثبيت الوكلاء في OpenClaw من دون التوقف عند الإعداد فقط

تثبيت OpenClaw لا يكفي. للوصول إلى وكيل ينجز عملاً حقيقيًا تحتاج إلى إكمال الإعداد الموجّه وتحديد الوكيل واختيار الإضافات المناسبة والتحقق من أول مهمة ملموسة.

الخطوة 1: الإعداد الموجّه وقاعدة مستقرة

تحتاج أولًا إلى قاعدة مستقرة. إذا تجاهلت هذه الخطوة فالوكلاء الذين ستنشئهم لاحقًا سيعتمدون على إعداد هش.

openclaw onboard --install-daemon
openclaw dashboard

الخطوة 2: أضف الوكيل الأول أو عرّفه

المهم ليس إنشاء عشرة وكلاء نظريين. المهم هو تعريف واحد بدور محدد ومهمة واضحة وحدود مفهومة.

openclaw agents add 
openclaw configure

الخطوة 3: استخدم الإضافات والأدوات فقط عندما تساعد

الإضافات تضيف قدرات عملية. والخطأ المعتاد هو تثبيت عدد كبير منها مبكرًا. الأفضل أن تبدأ بـ workflow مفيد واحد ثم تضيف فقط ما تحتاجه فعلًا.

openclaw plugins list
openclaw plugins install @openclaw/voice-call

قبل وبعد

  • ثبّت OpenClaw أولًا
  • ثم حدّد وكيلًا واحدًا واضحًا
  • ثم فقط أضف الإضافات والأدوات
