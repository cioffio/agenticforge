Windows

كيفية تثبيت OpenClaw على Windows من دون تعقيد غير لازم

الوثائق الرسمية توصي باستخدام OpenClaw على Windows عبر WSL2. هذا يتجنب كثيرًا من مشاكل التثبيت الأصلي ويحافظ على مسار أقرب إلى بيئة Linux التي تفترضها أغلب الشروحات.

المسار الموصى به: WSL2

إذا كنت تبدأ من الصفر، ثبّت WSL2 مع Ubuntu واعمل داخل هذا الطرفي. بعدها استخدم السكربت الرسمي:

curl -fsSL --proto '=https' --tlsv1.2 https://openclaw.ai/install.sh | bash
openclaw onboard --install-daemon
openclaw dashboard

المسار البديل: PowerShell المحلي

إذا أردت تجربة التثبيت المحلي على Windows فالأمر الرسمي في PowerShell هو:

قد يعمل للاختبارات السريعة، لكن WSL2 يبقى الخيار الأكثر استقرارًا للاستخدام الجدي.

iwr -useb https://openclaw.ai/install.ps1 | iex

مباشرة بعد التثبيت

  • شغّل الإعداد الموجّه.
  • افتح لوحة التحكم لاختبار الدردشة وواجهة التحكم.
  • إذا بدا شيء غير سليم، انتقل إلى إعداد أنظف داخل WSL2 بدلًا من تراكم الحلول المؤقتة داخل PowerShell.
